# Skill: Web Search
Esta habilidad permite al agente realizar búsquedas en internet para obtener información en tiempo real, noticias actualizadas o datos específicos que no están en su base de conocimientos interna.

## Cuándo usar
- Cuando el usuario pregunta por eventos actuales o noticias.
- Cuando se necesita verificar datos específicos o buscar precios.
- Cuando el usuario pide información que requiere acceso a la web.

## Herramientas
- `web_search`: Realiza una búsqueda en internet y devuelve los resultados más relevantes.
